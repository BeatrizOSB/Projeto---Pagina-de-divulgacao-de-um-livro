**Página Web - A Era Artificial**

*Descrição:*

Este código HTML cria uma página web simples para promover o livro *"A Era Artificial"*. A página apresenta:

*Título e descrição:*
Um título chamativo e uma descrição concisa do livro.
*Capa do livro:* Uma imagem da capa para visualização rápida.
*Sinopse:* Um resumo envolvente da história, incentivando o leitor a continuar lendo.
*Informações sobre o autor:* Uma breve biografia do autor, incluindo uma foto.
*Botões de compra:* Links diretos para a página de compra do livro na Amazon.

Este projeto foi feito por uma aluna em formação para colocar em prática o conteúdo aprendido em aula. Pode ser que hajam algumas coisas que poderiam ser feitas de uma maneira mais profissional, mas a ideia é estar em constante aprendizado e evolução!

*Como usar esse código:*

Baixe o projeto: Baixe todos os arquivos do projeto pra sua máquina.
Abra em um editor de código: Use um editor de código como o Visual Studio Code, Sublime Text ou qualquer outro que você preferir.
Dê uma olhada: Explore os arquivos e veja como o projeto foi estruturado.
Mude o que quiser: Sinta-se à vontade para mexer no código e fazer suas próprias modificações.
